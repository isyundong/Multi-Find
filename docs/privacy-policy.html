Privacy Policy for Multi Find Extension

Last Updated: December 2024

Information We Collect

Multi Find extension collects and stores the following information locally on your device:

Personal Preferences

  • Language settings for the extension interface
  • Search preferences (case sensitivity, whole word matching)
  • User interface customization settings

Search Data

  • Search keywords you enter
  • Search history within browser sessions
  • Current tab search states

How We Use Information

  • Functionality: To provide multi-keyword search and highlighting features
  • Personalization: To remember your language and search preferences
  • Performance: To improve search speed and user experience

Data Storage and Security

  • All data is stored locally on your device using Chrome's storage API
  • No data is transmitted to external servers
  • No personal information is shared with third parties
  • Data is automatically cleared when you uninstall the extension

Data Retention

  • Search keywords are temporarily stored during browser sessions
  • User preferences are stored until you change them or uninstall the extension
  • No long-term data collection or permanent storage of search content

Your Rights

  • You can clear all stored data by uninstalling and reinstalling the extension
  • You can modify your preferences at any time through the extension interface
  • No account creation or registration is required
